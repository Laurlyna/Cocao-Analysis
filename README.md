# Cacao Analysis Dashboard

## Overview
Analysis of global cacao and chocolate rating data using SQL and Tableau to uncover top performing companies, bean origins, and cocoa percentage trends.

## Tools Used
- MySQL — Data cleaning & Exploratory Data Analysis (EDA)
- Tableau — Interactive dashboard & data visualization

## Key Insights
- Tobago Estate (Pralus) is the highest rated company with a 4.0 average rating
- The average cocoa percentage across all chocolates is 71.69%
- Venezuela, Peru and Dominican Republic are among the top rated Bean Origin countries with a perfect 4.0 rating
- Chocolates with 70-80% cocoa percentage yield the highest average rating of 3.22, with ratings declining beyond 80%
